About

We know coding takes time and effort, but losing your code can take an instant. Git-break is a tool which helps to defend and protect your hard work. We offer 2 services: defend and check. Worried about your code being manipulated? We'll look out for you and send you an alert! Want to test other GitHub repos' security? Simply input the repo path with the check feature! With Git-break, you're ensured that your code is safe and in a trusted environment.
